# 📊 SaaS-Dashboard

A responsive, modern SaaS analytics dashboard built with vanilla JavaScript, HTML, CSS and Tailwind CSS. Designed for clarity and performance without the overhead of a JavaScript framework.

## Tech Stack

- HTML5 
- Tailwind CSS
- JavaScript
- CSS3

## ✨ Features 

- Displays Total Revenue, Active Users and Conversions
- Interactive monthly breakdown chart with 1-month and 1-year views
- Real-time activity stream for monitoring user and system events as they happen
- Displays order log showing customer, product and date
- At-a-glance progress toward quarterly targets
- Includes navigation for Analytics, Customers and Reports

## 🛠 Getting Started 

1. Clone the Repo
2. Open index.html in your browser
